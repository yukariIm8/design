Design

Iteration

Jun, 18, 2020

The test result taught me the "Motivational View" needs to be changed at the highest priority. When the user logs in, the "Motivational View" displays the app's selected image, which relates to the user's goals; however, I didn't explain it, so it confused users. Besides, it may display images that are different from the user's preference. In this case, it would confuse the user and make the user annoyed. Hence, I changed this feature from showing app selected images to presenting user-uploaded photos. Based on this new feature, I adjusted the app's flows and updated my app design as follows.
